# 006 文本编辑器

## Markdown 编辑器 - Typora



```bash
flatpak install flathub io.typora.Typora
```

推荐主题：[Phycat-neon](https://theme.typoraio.cn/theme/Phycat-Neon/)

黑色主题：**Phycat Abyss**

注：

主题安装目录 `/home/demodeom/.var/app/io.typora.Typora/config/Typora/themes`



## Visual Studio Code

[下载地址](https://code.visualstudio.com/Download)

下载 **x64**  **deb** 格式的安装包

```bash
sudo dpkg -i ~/下载/code_*.deb
```

看到以下提示，按 **Enter** 键

```
Add Microsoft apt repository for Visual Studio Code?                       
                                                                            
                   <是>                        <否>    
```
