# 更新、升级

## 01. 修改镜像

配置国内镜像，加快软件更新、安装速度。

1. 打开 **软件和更新**
2. 点击 **Ubuntu 软件** 选项卡
3. 点击 **下载自：** 后的下拉菜单，选择 **其他...**
4. 选择 **中国** 分类下的 **mirrors.aliyun.com**
5. 点击 **选择服务器**
6. 输入 **密码**， 点击**认证**
7. 点击 **关闭**按钮，点击 **关闭**按钮





## 02. 更新仓库

```bash
sudo apt update
```



## 03. 升级软件



```bash
sudo apt upgrade
```
