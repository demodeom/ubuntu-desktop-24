# 多媒体

## ffmpeg

可以安装常见的视频、音频解码器

```bash
sudo apt install ffmpeg
```

## 视频

### mpv

```bash
sudo apt install mpv
```



### vlc

```bash
sudo apt install vlc
```

## 音乐

### Amberol-本地音乐播放器

```bash
flatpak install flathub io.bassi.Amberol
```

### LX Music Desktop-在线音乐播放器

```bash
flatpak install flathub cn.toside.lxmusic.lx-music-desktop
```

### Gradia

如果你正在使用 GNOME 桌面环境，并且希望获得原生体验，那么 **Gradia** 是最新且非常出色的选择

```
flatpak install flathub be.alexandervanhee.gradia
```
