# Fcitx5

## 01. 安装 Fcitx5

```bash
sudo apt install fcitx5 fcitx5-chinese-addons fcitx-config-gtk
```

## 02. 开机启动 Fcitx5

1. 启动软件 **启动应用程序**
2. 点击 **添加** 按钮
3. 输入以下内容
   1. 名称： `fcitx5`
   2. 命令： `fcitx5`
   3. 注释： `fcitx5`
   4. 点击 **添加** 按钮
4. 点击 **关闭** 按钮

## 03. Fcitx5 主题配置

安装软件 **扩展管理器**

```bash
sudo apt install gnome-shell-extension-manager
```

启动软件 **扩展管理器**

点击 **浏览**

搜索 `Input Method Panel`, 点击 **安装** 按钮安装扩展 Input Method Panel



## 04. 重启系统

所有配置重启系统生效
